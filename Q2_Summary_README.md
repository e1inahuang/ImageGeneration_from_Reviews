# Q2 Analysis Summary - VICTGOAL Bike Helmet

## Overview
This section covers the GenAI-based analysis of the 582 customer reviews. We used sentence embeddings, clustering, and LLM-based feature extraction to understand the product's key attributes and sentiment.

## Files Generated
- **`bike_helmet_reviews_analysis.ipynb`**: The Jupyter notebook containing the complete analysis workflow.
- **`Final_Project_Q2.docx`**: The final report document for Q2, including methodology, cluster analysis, and extracted features.
- **`cluster_distribution.png`**: A bar chart visualization of the 8 review clusters.
- **`helmet_extracted_features.json`**: JSON file containing the structured features extracted from reviews (Design, Materials, Key Features, etc.).
- **`cleaned_helmet_reviews.csv`**: The preprocessed reviews used for analysis.

## Key Findings
- **8 Distinct Topics** identified, ranging from "Safety/Light" to "Fit/Comfort" and "Visor/Goggles".
- **Visual Features Extracted**:
    - **Style**: Modern, sporty, "Robocop-like".
    - **Key Components**: Magnetic detachable goggles, USB rechargeable rear LED light.
    - **Materials**: PC shell, EPS foam, matte finish.
- **Image Generation**: A prompt was generated based on these features to create a product visualization (Note: Image generation subject to API availability).
